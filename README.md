# 📈 Data-Driven Stock Analysis — Nifty 50 Dashboard

A full end-to-end data analytics project that takes raw YAML stock data,
cleans it, loads it into MySQL, and presents it through an interactive
Streamlit dashboard and a Power BI report.

**Pipeline:**  YAML files → 50 CSVs → MySQL Database → Streamlit Dashboard + Power BI

---

## 📁 Project Structure

```
stock_analysis/
│
├── extract_yaml_to_csv.py      # Step 1 — Reads all YAML files → 50 per-symbol CSVs + master CSV
├── clean_data.py               # Step 2 — Cleans data, adds computed columns (daily_return, month, etc.)
├── db_upload.py                # Step 3 — Uploads cleaned CSV to MySQL via SQLAlchemy
│
├── data/
│   ├── csv/                    # 50 individual stock CSVs (auto-generated by Step 1)
│   ├── master_stock_data.csv   # Combined raw data (auto-generated by Step 1)
│   ├── cleaned_stock_data.csv  # Final clean data used by dashboard (auto-generated by Step 2)
│   └── nifty50_sectors.csv     # Sector mapping for all 50 Nifty stocks
│
├── app/
│   └── app.py                  # Streamlit interactive dashboard (8 visualizations)
│
├── PowerBI_Guide.txt           # Step-by-step Power BI instructions for this project
├── requirements.txt            # Python dependencies
└── README.md                   # This file
```

---

## ⚙️ Prerequisites

- Python 3.10 or higher
- MySQL Server installed and running locally
- Power BI Desktop (free download from microsoft.com)

---

## 🚀 Setup & Execution — Step by Step

### Step 1 — Install Python dependencies

Open a terminal/command prompt inside the `stock_analysis` folder and run:

```bash
pip install -r requirements.txt
```

---

### Step 2 — Configure your MySQL password

Open `db_upload.py` in any text editor and find this block near the top:

```python
DB_CONFIG = {
    "host":     "localhost",
    "port":     3306,
    "user":     "root",
    "password": "mohan",       # ← change this to your actual MySQL password
    "database": "nifty50_db",
}
```

Save the file after editing.

---

### Step 3 — Run the data pipeline

Run the three scripts in order from the `stock_analysis` folder:

```bash
# Step 3a — Extract YAML files into individual CSVs
python extract_yaml_to_csv.py

# Step 3b — Clean and enrich the data
python clean_data.py

# Step 3c — Upload to MySQL database
python db_upload.py
```

Each script will print confirmation messages when done.
After Step 3c, your MySQL database `nifty50_db` will have two tables:
- `stock_prices` — all cleaned OHLCV data with computed columns
- `sector_mapping` — sector labels for each of the 50 stocks

---

### Step 4 — Launch the Streamlit Dashboard

```bash
streamlit run app/app.py
```

This opens the dashboard in your browser at `http://localhost:8501`

---

### Step 5 — Build the Power BI Dashboard

See `PowerBI_Guide.txt` in this folder for full step-by-step instructions.

## 📊 Streamlit Dashboard — Visualizations

| # | Section | What it Shows |
|---|---------|---------------|
| 1 | KPI Summary Cards | Green/Red stock count, avg price, avg volume, best & worst return |
| 2 | Top 10 Green & Red | Horizontal bar charts of the year's top gainers and losers |
| 3 | Volatility Analysis | Top 10 most volatile stocks (std dev of daily returns) |
| 4 | Cumulative Return | Line chart tracking top 5 performers across the full year |
| 5 | Sector-wise Performance | Average yearly return grouped by industry sector |
| 6 | Correlation Heatmap | How closely different stocks move together |
| 7 | Monthly Gainers & Losers | 12 bar charts — top 5 up and down stocks for every month |
| 8 | Candlestick + Volume | Interactive OHLCV chart for any individual stock |

Use the **sidebar** to filter by Sector, Stock, and Date Range — all charts update live.

## 📌 Notes

- YAML source files must be in monthly folders named `2023-10/`, `2023-11/`, etc., one level above `stock_analysis/`. The pipeline auto-discovers all such folders.
- All file paths in the scripts are relative to script location — the project works on Windows, Mac, and Linux without changes.
- `cleaned_stock_data.csv` is the single source of truth for both the Streamlit app and Power BI.
